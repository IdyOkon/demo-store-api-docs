## Overview
The Demo Store API is a public REST API that provides sample e-commerce data such as products, categories, and carts.
This documentation demonstrates how to clearly describe API functionality for developer integration.

## Base URL
https://fakestoreapi.com

## Authentication
The API uses token-based authentication (JWT) for protected endpoints.
Tokens are obtained via the login endpoint and must be included in the `Authorization` header.
